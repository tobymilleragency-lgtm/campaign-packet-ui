# Landing Page Spec — Export Only

This tab generates the landing-page handoff spec. It does not deploy a site. Actual site build goes through the normal site-build flow.

## Page goal
Convert homeowners into free project review requests.

## Hero
Headline: Get Honest Remodel Advice Before You Hire
Subhead: Planning a bathroom remodel? Relax Remodel Consulting helps you review the project, clarify the right questions, and move forward with more confidence before choosing a contractor.
Primary CTA: Request a Free Project Review
Secondary note: Advisory help only — not a contractor estimate.

## Sections
1. Hero with advisory positioning
2. Common homeowner concerns
3. What the project review covers
4. What this is / what this is not
5. Simple request form
6. Trust and privacy note
7. Thank-you next step

## Form fields
- Full name
- Phone
- Email optional
- City / ZIP
- Project type
- What are you considering?
- Have you already received a contractor bid? yes/no
- Best time to contact

## Trust copy
Relax Remodel Consulting is here to help you understand the project before you make a hiring decision. This is not a contractor estimate and does not obligate you to hire anyone.

## Thank-you page copy
Thanks — your project review request was received. We will review what you shared and follow up with next steps after the request is checked. SMS/CRM/AI follow-up is only treated as live after A2P verification is complete.

## Handoff notes
- Use advisory language throughout.
- Do not add “instant text” claims unless A2P is verified.
- If GHL webhook is added, use the payload spec from the GHL Webhook Export tab.
