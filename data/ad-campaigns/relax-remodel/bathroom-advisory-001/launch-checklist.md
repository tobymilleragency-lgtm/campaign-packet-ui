# Launch Checklist

## Positioning
- [ ] Campaign says Relax Remodel Consulting is the homeowner advisor, not a contractor.
- [ ] Offer says “free project review / honest advice before you hire.”
- [ ] No contractor-estimate language appears in ads, page, form, or follow-up.

## Creative
- [ ] AI images are lifestyle/concept only unless clearly based on real provided project photos.
- [ ] No AI-generated image implies completed client work.
- [ ] Creative is believable, not luxury-fake or over-polished.

## Landing page handoff
- [ ] Landing-page spec exported for normal site-build flow.
- [ ] Form fields match the webhook spec.
- [ ] Thank-you copy avoids instant-response claims unless verified.

## GHL handoff
- [ ] Webhook spec exported for browser/GHL agent.
- [ ] GHL lead received from a real test submission.
- [ ] Tags/source fields verified in GHL.

## A2P / CRM / AI follow-up guardrail
- [ ] SMS/CRM/AI follow-up only marked live after A2P is verified — no “instant text” claims before then.
- [ ] If A2P is not verified, SMS automations are disabled or not claimed publicly.
- [ ] Any AI follow-up claims are backed by live workflow proof, not assumed.

## Tracking and approval
- [ ] Mobile layout checked.
- [ ] Form tested end-to-end.
- [ ] Privacy/consent language checked.
- [ ] Ad copy approved.
- [ ] Creative approved.
- [ ] Launch owner and date recorded.
